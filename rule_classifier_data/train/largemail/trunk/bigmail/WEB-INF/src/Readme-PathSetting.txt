---------------------------------------------------------------------------------------
                     JRE 1.5.0_*の環境では（XML形式）
---------------------------------------------------------------------------------------
アップロードフォルダ指定し次のどちらでも使用できます。
　　　Windowsサーバーならば、
　　　　C:\bigmail\upload　あるいは　C:/bigmail/upload
      linux/unixサーバーならば、
　　　　/usr/bigmail/upload
      のように'/'でパスを区分します。

臨時アップロードフォルダ（temp dir)の場合はWindows,Linux系いずれも
'/'を使かってください。
　　　例）　C:/bigmail/upload　（Windows), /usr/bigmail/temp (linux系）　

---------------------------------------------------------------------------------------
                     JRE 1.4.2_*の環境では（key=value形式）
---------------------------------------------------------------------------------------
JRE1.4.2バージョンではアップロードフォルダ指定し少し注意する必要があります。
　　　Windowsサーバーでは'\'を使う時、必ずエスケープ文字('\')を入れなければ
　　なりません。　つまり、この形となります。
　　　例） C:\\bigmail\\upload
      linux系のサーバーでは上記のように設定すればよいでしょう。

要するに、'\'より'/'を使うのが覚えやすいし、もし発生しかねないバグからも安全です。
